{$P8LANG['allow_admin_ip']}
是否启用后台IP允许: 不开启 启用仅限局域网访问 开启 启用IP允许后，只有允许的IP才能访问
允许IP: 注意:访问的客户端必须是固定IP或者在固定的IP段内
否则很有可能导致客户端无法正常访问,此时只能通过修改后台文件强制进入后台进行修改恢复
书写格式如下：
192.168.0.2
192.168.0.5
...
允许IP段: 每个段位一行，书写格式如下：
192.168.0.2-192.168.0.255
192.168.1.0-192.168.254.255
允许IP段内例外IP: 附注:在允许IP段内，可以限制排除下列IP地址
书写格式如下：
192.168.0.2
192.168.0.5
...
