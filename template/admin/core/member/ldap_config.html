1、如果你的php已经有 php_ldap.dll文件，打开php.ini文件， 找到 “;extension=php_ldap.dll” 去掉分号后保存。
2、如果没有 php_ldap.dll文件，你可以从网上找的符合自己php版本的 php_ldap.dll 文件，将文件放在相应的目录，重复上过程。
3、重启Apache或Nginx服务。

二、Linux下LDAP安装
1、如果你的php是编译安装的，那么你需要加“--with-ldap”参数，重新编译安装。
在php安装包下的 ext/ldap 目录下执行以下指令：
/usr/local/php/bin/phpize
./configure --with-php-config=/usr/local/php/bin/php-config --with-ldap
make
make install
2、如果你的php是通过Linux的包管理器安装的，你可以通过包管理器安装php_ldap。
以Debian为例，执行apt-get install php7.0-ldap或yum install php7.0-ldap
3、重启Apach或Nginx服务。
