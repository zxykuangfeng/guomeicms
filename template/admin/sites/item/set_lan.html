{$P8LANG['config']}(对所有站点生效) 参数备注
应用场景说明 1、当需要设置某个时间节点（如3年）前的内容为局域网访问。
2、时间节点前的信息在列表页面将不显示。
3、时间节点前的信息不再生成静态页面。
4、确需启用的，请务必备份数据后按流程1-7操作。
站群是否开启 默认不启用 默认不启用，需要设置时间节点前的内容为局域网才开启
时间节点 对所有站点生效，时间节点前的信息在列表页面将不显示，不再生成静态页面
操作配置      {$P8LANG['html_unset_note']}
步骤1：例外栏目设置 查看和设置全部站点列外栏目设置>> 例外栏目将不受限局域网访问规则设置的影响，如有必要，请进入各个站点的配置后再操作
步骤2：批量设置限局域网访问 {$P8LANG['html_init_note']} 对所有站点生效
步骤3：更新栏目内容统计 更新设置局域网化后栏目的内容数统计（真实数据以查看后台栏目为准） 对所有站点生效
步骤4：更新栏目缓存 对所有站点生效
步骤5：删除限定内容静态页面 建议备份/html文件夹，仅删除限定内容静态页面 对所有站点生效
其它步骤 步骤5：进入各个分站，重新静态化所有栏目
步骤6：进入各个分站，重新静态化所有内容
步骤7：进入各个分站，重新静态化首页 		进入各个分站后分别操作
